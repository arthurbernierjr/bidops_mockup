# To Start App

```
cd api
bundle
rails db:create
rails db:migrate
rails db:seed
rails s -p 3006
```
## In second terminal
```
cd frontend
npm i
npm run watch
```

### App is visible at localhost:3000

### Tech Stack
1. Kofu JS Self Created Object Oriented Front End Framework Built in CSS-in-JS Support
1. Webpack, Gulp, Babel, CoffeeScript(dev server only)
1. Ruby On Rails
1. PostgreSQL
